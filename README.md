# ShoppyGlobe REST API

## Project Description
This is a REST API built using Node.js, Express.js, and MongoDB Atlas for managing products, carts, and user authentication.

## Github 


## Features

- Product CRUD APIs
- Cart Management APIs
- JWT Authentication
- MongoDB Atlas Integration
- Thunder Client Testing

## Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- Thunder Client

## API Endpoints

### Products

GET /products
GET /products/:id
POST /products
PUT /products/:id
DELETE /products/:id

### Users

POST /register
POST /login

### Cart

POST /cart
PUT /cart/:id
DELETE /cart/:id
